    Log_Code Details (v001)

    Explanation:

    The log_code property is used to describe the type of log generated by the fixture.

    Data Structure:

    2 bytes (number, big-endian)

    Info Logs (Notes):

    • Fixture power on: Log generated when the fixture is powered on.
    • Log throttled: Log generated when the log is throttled. This indicates that the fixture as already created 20 successive identical logs and will not create another log until the log changes.
